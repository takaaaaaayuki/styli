## ✨ アプリについて

**Styli** は、ファッションにおける「似合う」が分からない課題を解決する、**服管理システム & AIコーディネート提案アプリ**です。  
Google Cloud Vision APIを活用して服の特徴を自動認識し、登録アイテムの色・季節・TPOなどをもとに最適なコーディネートを提案。さらに他ユーザーとのSNS的な交流機能も備えています。

## ⚙️ コーディネートAIの仕組み

1. **色相の相性評価**  
   - 色相環の距離をベースに、調和スコアを算出（類似色・補色・中差色など）

2. **TPO・季節整合性のスコア化**  
   - 入力条件とアイテム属性をマッチングし、整合性を評価

3. **ユーザー登録状況に応じた柔軟性**  
   - 登録アイテムが少ない場合は条件を緩和し、提案が必ず表示される工夫を実装

4. **スコアの重み付け**  
   - 色相 / TPO / 季節の3軸スコアにそれぞれ重みをかけ、最終スコアでマッチング
